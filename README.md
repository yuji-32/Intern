# インターントラッカー

インターンの応募先の企業情報と選考予定を、  
一元的に管理できるWebアプリです。

🔗 https://intern-alpha-seven.vercel.app/

---

## アプリ概要

就活では、企業ごとの応募状況・締切・面接予定を同時に管理する必要がありますが、  
情報が分散しやすく、抜け漏れが起きやすいと感じました。

そこで、  
**企業管理とスケジュール管理を一体化したアプリ**として本アプリを制作しました。

---

## 主な機能

- 企業情報の登録・編集・削除
- 応募状況（ステータス）の管理
- カンバン形式での進捗管理
- 面接・締切などのスケジュール管理
- 企業と予定の紐付け
- 応募サイトの入力補助（過去入力から候補表示）
- LocalStorageによるデータ保存

## 工夫した点

### 1. 企業情報と予定の紐付け

企業と面接・締切などの予定を関連付けて管理できるようにしました。  
企業一覧から次の予定を確認できるため、  
「何をするべきか」がすぐ分かる設計にしています。

---

### 2. カンバン形式による進捗管理

複数の企業の選考状況を直感的に把握できるよう、  
ステータスごとに並べるカンバンUIを採用しました。  
進捗の可視化によって、管理のしやすさを向上させています。

---

### 3. 入力負担を減らす工夫

応募サイト名などは、一度入力すると候補として表示され、  
次回以降は選択するだけで入力できるようにしました。  
入力の手間を減らし、継続して使いやすいようにしています。

---

### 4. 情報の優先度を意識したUI

企業カードでは、

- 会社名
- 次の予定
- ステータス・締切

を優先的に表示し、  
一目で状況が把握できるようにしました。

## 制作背景

就活では、企業情報とスケジュールを別々に管理することが多く、  
情報の整理に手間がかかると感じました。

本アプリでは、  
**企業管理と選考管理を一体化することで、効率的に就活を進められるようにすること**を目的としています。
